Having an issue? Let us know!

To report a problem, please send an email to lostintime101dev@gmail.com with the following details:

📌 For Question-Related Issues
  • 🔹 Question number: {{ question['id'] }}
  • 🔹 A brief description of the issue and why the answer is incorrect.
⚙️ For Technical Issues
  • 🔹 Steps to reproduce the error.
  • 🔹 A screenshot of the issue (use Excalidraw to annotate if needed).
  • 🔹 Your device model (for mobile users).

Alternatively, you can create an issue on GitHub or even fork the project, fix the bug, and submit a pull request!
